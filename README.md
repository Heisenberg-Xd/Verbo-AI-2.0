# VerboAI
VerboAI is a multilingual AI Intelligence Platform designed to transform large collections of documents into structured insights and searchable knowledge.

🚀 Overview

VerboAI is an advanced AI-powered intelligence platform designed to analyze large collections of multilingual documents and convert them into structured knowledge and actionable insights.

The system uses modern Natural Language Processing and Machine Learning techniques to automatically detect languages, translate documents, generate embeddings, cluster topics, extract keywords, perform sentiment analysis, and organize information into a searchable knowledge base.

With integrated AI chat (RAG) capabilities, users can interactively ask questions about uploaded documents and retrieve accurate answers grounded in their data.

✨ Key Features
🌍 Multilingual Intelligence

Automatically detects document languages and translates them into English for unified analysis.

📂 Document Processing Pipeline

Upload multiple documents and run them through an AI intelligence workflow.

🧠 Semantic Understanding

Uses Sentence-BERT embeddings for deep semantic analysis.

🧩 Topic Clustering

Groups documents into clusters using KMeans clustering.

📝 Automatic Summaries

Generates concise summaries for each cluster.

🔑 Keyword Extraction

Extracts top keywords using TF-IDF.

😊 Sentiment Analytics

Analyzes document tone using VADER sentiment analysis.

📊 Cluster Visualization

Visualizes document clusters using PCA dimensionality reduction.

🗂 Knowledge Base Generation

Automatically organizes documents into structured topic folders.

📑 Intelligence Reports

Generates detailed reports containing insights, statistics, and analytics.

🤖 AI Chat with Knowledge Base (RAG)

Ask questions about documents and receive contextual AI answers.
