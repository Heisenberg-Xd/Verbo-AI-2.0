       Machine Learning – Introduction
       Machine Learning - Introduction
=======================================

1. Machine learning is a branch of artificial intelligence that learns from data. 2. In this, the computer is given the ability to learn without explicit programming. 3. The main objective of machine learning is to recognize patterns and make predictions. 4. It is of three types: Supervised, Unsupervised, and Reinforcement Learning. 5. In Supervised Learning, the model is trained on labeled data. 6. In Unsupervised Learning the model finds patterns in the data without labels. 7. In Reinforcement Learning the agent learns on the basis of rewards and punishments. 8. Linear Regression is a simple machine learning algorithm that predicts numerical values. 9. Decision Tree algorithm divides the data into a tree structure. 10. Random Forest is a combination of multiple Decision Trees which gives better results. 11. Neural Networks solve complex problems by mimicking the human brain. 12. Neural Networks with many layers are used in Deep Learning. 13. Overfitting occurs when the model misses the training data too much. 14. Cross-Validation is an important technique to measure the accuracy of the model. 15. Feature Engineering is the process of extracting useful features from data. 16. Python and R are the most popular programming languages ​​for machine learning. 17. Scikit-learn, TensorFlow and PyTorch are the major ML libraries. 18. Machine learning is used in health, finance, education and entertainment. 19. Spam filters, face recognition, and recommendation systems are examples of ML. 20. Machine learning has revolutionized medical diagnosis. 21. Big Data and Cloud Computing have made machine learning more powerful. 22. Data quality plays the most important role in the success of a machine learning model. 23. In the future, machine learning will bring automation in every field. 24. Learning Machine Learning has become an important skill in today's era.
